## Requirements:

```
Spring Boot: 3.5.6
Maven: 3.9+
Java: 24
```

### Clone this repository:

```bash
git clone https://github.com/deepaksorthiya/spring-boot-oauth2-authserver.git
cd spring-boot-oauth2-authserver
```

### Start Spring Boot Service:

```bash
./mvnw spring-boot:run
```
